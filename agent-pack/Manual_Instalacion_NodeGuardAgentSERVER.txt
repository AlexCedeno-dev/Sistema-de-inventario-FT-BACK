# MANUAL DE INSTALACIÓN - NODEGUARD AGENT SERVER

Después de descomprimir el archivo ZIP, dentro de la carpeta se debe ver algo similar a esto:

* NodeGuard
* Tools
* 1_INSTALAR_NODEGUARD.bat
* install-nodeguard.ps1
* Manual_instalacion_NodeGuardSERVER.txt
* NodeGuardAgentSERVER.exe

## PASO 1: DESCOMPRIMIR LA CARPETA

1. Descargar el paquete desde la página de Agentes.
2. Dar clic derecho sobre el archivo ZIP.
3. Seleccionar Extraer todo.
4. No importa dónde se descomprima, puede ser en Descargas, Escritorio o cualquier carpeta temporal.

## PASO 2: ABRIR POWERSHELL COMO ADMINISTRADOR

1. Presionar Inicio.
2. Buscar PowerShell.
3. Dar clic derecho.
4. Seleccionar Ejecutar como administrador.

Después, entrar a la ruta donde se descomprimió la carpeta.

Ejemplo si está en Descargas:

cd "$env:USERPROFILE\Downloads\NodeGuardAgentSERVER_Pack"

Ejemplo si está en Escritorio:

cd "$env:USERPROFILE\Desktop\NodeGuardAgentSERVER_Pack"

Nota:
La ruta puede cambiar dependiendo de dónde se haya descomprimido el ZIP.

## PASO 3: COPIAR LAS CARPETAS AL DISCO LOCAL C:

Desde la carpeta descomprimida, copiar estas carpetas:

* NodeGuard
* Tools

Y pegarlas directamente en:

C:\

Al final deben existir estas rutas:

C:\NodeGuard
C:\Tools

Dentro de C:\NodeGuard debe estar el agente:

C:\NodeGuard\NodeGuardAgentSERVER.exe

Dentro de C:\Tools debe estar NSSM:

C:\Tools\nssm\nssm.exe

## PASO 4: ABRIR EL SCRIPT INSTALL-NODEGUARD.PS1

Dentro de la carpeta descomprimida, abrir el archivo:

install-nodeguard.ps1

Puede abrirse en Bloc de notas. Eso es normal.

## PASO 5: COPIAR Y EJECUTAR EL SCRIPT

1. Copiar todo el contenido del archivo install-nodeguard.ps1.
2. Ir a la ventana de PowerShell abierta como administrador.
3. Pegar todo el contenido.
4. Presionar Enter.
5. Esperar a que termine la instalación.

Al final debe aparecer el servicio con estado:

Running

Si aparece Running, la instalación fue correcta.

## VALIDACIÓN FINAL

Para confirmar que quedó instalado correctamente:

1. Abrir Administrador de tareas.
2. Ir a la pestaña Detalles.
3. Buscar:

NodeGuardAgentSERVER.exe

También se puede validar en PowerShell con:

Get-Service NodeGuardAgentSERVER

Debe aparecer como:

Running

## COMANDOS ÚTILES

-Reiniciar el agente:

Restart-Service NodeGuardAgentSERVER

-Detener el agente:

Stop-Service NodeGuardAgentSERVER

-Iniciar el agente:

Start-Service NodeGuardAgentSERVER

-Validar estado del agente:

Get-Service NodeGuardAgentSERVER

-Forzar cierre del proceso si se queda trabado:

Stop-Process -Name NodeGuardAgentSERVER -Force -ErrorAction SilentlyContinue

Después iniciar nuevamente:

Start-Service NodeGuardAgentSERVER

## ELIMINAR EL AGENTE

Para borrar el agente de la computadora, abrir PowerShell como administrador y ejecutar:

Copear todo apatrir de aqui 

$SERVICE = "NodeGuardAgentSERVER"
$NSSM = "C:\Tools\nssm\nssm.exe"

Write-Host "Eliminando NodeGuardAgentSERVER..." -ForegroundColor Cyan

# Detener servicio si existe
Stop-Service $SERVICE -Force -ErrorAction SilentlyContinue

# Eliminar servicio
if (Test-Path $NSSM) {
    & $NSSM remove $SERVICE confirm
} else {
    sc.exe delete $SERVICE
}

Start-Sleep -Seconds 2

# Borrar archivos
Remove-Item "C:\NodeGuard" -Recurse -Force -ErrorAction SilentlyContinue

Write-Host "Validando..." -ForegroundColor Cyan
Get-Service $SERVICE -ErrorAction SilentlyContinue
Get-Process NodeGuardAgentSERVER -ErrorAction SilentlyContinue

Write-Host "Proceso terminado. Si no apareció nada en la validación, ya quedó eliminado." -ForegroundColor Green
