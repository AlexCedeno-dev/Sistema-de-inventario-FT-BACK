# GUÍA DE INSTALACIÓN - NODEGUARD AGENT

Objetivo:
Instalar el agente NodeGuard en una computadora Windows para que se ejecute en segundo plano como servicio de Windows.

El agente debe quedar:

* Ejecutándose automáticamente con Windows.
* Sin abrir ventana de CMD.
* Visible en Administrador de tareas.
* Visible como servicio de Windows.
* Reportando al dashboard de NodeGuard.

IMPORTANTE:
Este procedimiento debe realizarlo únicamente personal autorizado del área de TI.

## ARCHIVOS DEL PAQUETE

Al descargar y descomprimir el ZIP, la carpeta debe contener archivos similares a estos:

NodeGuardAgentSERVER_Pack
│
├─ NodeGuardAgentSERVER.exe
├─ install-nodeguard.ps1
├─ Manual_Instalacion_NodeGuardAgentSERVER.txt
└─ tools
└─ nssm.exe

Nota:
El nombre del agente puede variar dependiendo de la versión, por ejemplo:

NodeGuardAgentSERVER.exe
NodeGuardAgent.exe

Usar siempre el archivo .exe que viene dentro del paquete descargado.

## PASOS DE INSTALACIÓN

1. Descargar el paquete desde la página de Agentes

Desde la página de Agentes de NodeGuard, dar clic en el botón para descargar el paquete del agente.

Se descargará un archivo .ZIP.

2. Descomprimir el archivo ZIP

Dar clic derecho sobre el archivo .ZIP y seleccionar:

Extraer todo

No importa en qué carpeta se descomprima, puede ser en Descargas, Escritorio u otra ubicación temporal.

IMPORTANTE:
No ejecutar los archivos directamente desde el ZIP. Primero se debe descomprimir.

3. Ejecutar el instalador como administrador

Dentro de la carpeta descomprimida, buscar el archivo:

install-nodeguard.ps1

Dar clic derecho sobre el archivo y seleccionar:

Ejecutar con PowerShell

Si Windows permite ejecutarlo correctamente, continuar con el proceso.

Nota:
Es normal que la ventana de PowerShell se cierre después de ejecutar una parte del proceso.

4. Si el archivo se abre en Bloc de notas

Si al dar clic en install-nodeguard.ps1 se abre el Bloc de notas, hacer lo siguiente:

1. Abrir PowerShell como administrador.
2. Abrir el archivo install-nodeguard.ps1 con Bloc de notas.
3. Copiar todo el contenido del script.
4. Pegar el contenido en PowerShell.
5. Presionar Enter.

Para abrir PowerShell como administrador:

1. Presionar Inicio.

2. Buscar PowerShell.

3. Clic derecho.

4. Seleccionar Ejecutar como administrador.

5. Validar que se crearon las carpetas

Ir al disco local C: y verificar que existan estas carpetas:

C:\Tools
C:\NodeGuard

Si ambas carpetas existen, continuar con el siguiente paso.

6. Copiar el agente a la carpeta NodeGuard

Copiar el archivo del agente que viene dentro del ZIP, por ejemplo:

NodeGuardAgentSERVER.exe

O según la versión:

NodeGuardAgent.exe

Y pegarlo dentro de esta carpeta:

C:\NodeGuard

La ruta final debe quedar similar a:

C:\NodeGuard\NodeGuardAgentSERVER.exe

O:

C:\NodeGuard\NodeGuardAgent.exe

IMPORTANTE:
No cambiar el nombre del archivo .exe, a menos que el script esté configurado para ese nombre.

7. Ejecutar nuevamente el instalador

Después de copiar el .exe en C:\NodeGuard, ejecutar nuevamente el archivo:

install-nodeguard.ps1

Se puede hacer de cualquiera de estas dos formas:

Opción A:
Clic derecho sobre install-nodeguard.ps1 y seleccionar Ejecutar con PowerShell.

Opción B:
Abrir PowerShell como administrador, copiar el contenido del script, pegarlo y presionar Enter.

Esperar a que termine la instalación.

Al final debe aparecer el servicio con estado:

Running

Si aparece Running, la instalación fue correcta.

8. Validar en Administrador de tareas

Abrir el Administrador de tareas:

Ctrl + Shift + Esc

Ir a la pestaña:

Detalles

Buscar el proceso del agente:

NodeGuardAgentSERVER.exe

O según la versión:

NodeGuardAgent.exe

Si aparece en la lista, el agente está activo.

9. Validar en servicios de Windows

Presionar:

Win + R

Escribir:

services.msc

Buscar el servicio:

NodeGuardAgentSERVER

Debe aparecer como:

En ejecución

10. Validar en el dashboard

Entrar al dashboard de NodeGuard y confirmar en la pagina de "Agentes" que el equipo aparezca como conectado o detectado.


## COMANDOS ÚTILES

Ver estado del servicio:

Get-Service NodeGuardAgentSERVER

Reiniciar servicio:

Restart-Service NodeGuardAgentSERVER

Detener servicio:

Stop-Service NodeGuardAgentSERVER

Iniciar servicio:

Start-Service NodeGuardAgentSERVER

Ver errores del agente:

Get-Content "C:\NodeGuard\logs\error.log" -Tail 50

Ver salida normal del agente:

Get-Content "C:\NodeGuard\logs\out.log" -Tail 50

## ERRORES COMUNES

1. El script se abre en Bloc de notas

Solución:
Abrir PowerShell como administrador, copiar el contenido del script y pegarlo directamente en PowerShell.

2. El servicio no queda en Running

Revisar que el .exe esté dentro de:

C:\NodeGuard

También revisar el log:

C:\NodeGuard\logs\error.log

3. No aparece en Administrador de tareas

Revisar que el servicio esté iniciado:

Get-Service NodeGuardAgentSERVER

Si aparece detenido, ejecutar:

Start-Service NodeGuardAgentSERVER

4. No aparece en el dashboard

Revisar:

* Que la computadora tenga red.
* Que el servidor de NodeGuard esté encendido.
* Que el agente tenga la URL correcta del servidor.
* Que no esté bloqueado por firewall.
* Revisar los logs en C:\NodeGuard\logs.
